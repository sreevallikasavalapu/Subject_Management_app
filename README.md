#📚 Subject Management App
A web application that allows users to add, edit, and delete subjects dynamically.

🚀 Features
✅ Add new subjects
✅ Edit existing subjects
✅ Delete subjects
✅ User-friendly UI
✅ Responsive design

🖥️ Tech Stack
Frontend: React.js, Bootstrap

Backend: .NET Web API

Database: SQL Server

Version Control: Git & GitHub
